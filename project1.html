CPRE 281 - Final Project

The Calculator

This class was an introduction to the subject of hardware design. During the semester, we worked with a language named Verilog to program an FPGA, and we uses software name Hades to simulate logic circuits.

I and my partner decided to create a calculator. To do this, I first designed the logic of the calculator using our understanding of boolean algebra. We then proceeded to create the circuit logic using logic gates and basic memory structures.

The software allowed us to create modules of functionality and build up to more and more complex objects. In this way we were able to create controller and memory objects that allowed for addition, subtraction, multiplication, and value storing. The input was done through the use of buttons, and the results were displayed on simple LCD displays.

For the final presentation, we recorded a screencast and edited a video explaining the functionality of our project. We then uploaded this video to YouTube.
